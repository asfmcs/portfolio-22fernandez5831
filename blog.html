HTML

As a student, programming is something that have always caught my attenton. How websites work and the "magic" behind it. Very luckily, I go to a school (American School Foundation of Monterrey) where I am offered a computers class that teaches programming!! Over the course of this past month, i've learned so much about how websites work and even made a few. So far i'm having so much fun and each day learn new codes and tricks of programming.

Here are a few of the very basic HTML codes for creating your first website!

Tag Function
!DOCTYPE Defines the document type
html Defines an HTML document
head Defines information about the document
title Defines a title for the document
body Defines the document's body
h1 to h6 Defines HTML headings
p Defines a paragraph
br Inserts a single line break
id="name-of-choice" A unique element with the specified id value.
class="intro important" The class attribute specifies one or more classnames for an element.

Note: it is importaint that at the begginig of each tag you add a "<" and an ">" after. Then, to close it up (after writing the data it will contain)you will have to write a / before each tag.

CSS

For me personally, CSS was one of my favorite parts of making a website. With CSS one is able to make their websites more interactive and visually attractive. Coding in CSS is quite different than HTML. First off, you have to open a new section that marks where you begin your CSS. To do this you have to create a < style > tab. Then, add a selector (this can be an id or class) which identifies what it is that you want to style. After this, you are able to chose from a very broad selection of options.


Every Cascading Style Sheet (whether it is contained in a .css file, or embedded in the head element of an HTML document) is a series of instructions called statements. A statement does two things: it identifies the elements in an HTML document that it affects. Westciv.com
